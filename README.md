# Responsive Image Gallery

This project is developed as part of a GitHub Internship Task.

It is a fully responsive and interactive image gallery built using HTML, CSS, and JavaScript. The gallery allows users to view images in a structured layout, apply category-based filters, and open images in a lightbox with navigation controls.

## Features
- Responsive grid layout for different screen sizes
- Image filtering by categories (All, Nature, Turkey, Islamabad)
- Lightbox view for enlarged images
- Next and Previous navigation buttons
- Smooth hover effects and transitions
- Modern and user-friendly interface

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- JavaScript (DOM Manipulation)

## How to Use
1. Open the project in a web browser
2. Use the category buttons to filter images
3. Click on any image to open it in lightbox view
4. Navigate using next/previous buttons
5. Close the lightbox using the close button
